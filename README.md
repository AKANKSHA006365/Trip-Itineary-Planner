# Trip Itinerary Planner

A comprehensive web application for planning travel itineraries with real-time route information.

## Features

- 🔐 User Authentication (Login/Registration)
- 📝 Forgot Password Functionality
- 🗺️ Travel Itinerary Generation (1-6 days)
- 🚗 Real-time Route Information (Flight, Train, Bus, Car)
- 📍 50+ Indian Cities Supported
- 🎨 Modern, Responsive UI

## Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **APIs:** OpenRouteService (for routes)
